# Volgrei

**The execution workspace for freelancers.**
Tasks, notes, and AI — in a single iOS app.

🟢 Currently in active development · iOS · launching soon

---

### What it is

Volgrei is an iOS productivity app designed around how freelancers actually work. Not a generic todo list — a focused execution layer that combines structured task management, contextual notes, and an integrated AI agent named Grei. Everything in one place, nothing extra.

---

### Core features

| | |
|---|---|
| **Task management** | Structured, deadline-aware task system built for freelance workflows. |
| **Smart notes** | Contextual notes that live alongside your work, not in a separate app. |
| **Grei — AI agent** | An integrated AI assistant that understands your workspace and helps you move faster. |
| **Social layer** | Connect with other freelancers. Share progress, stay accountable. |

---

### Built with

`Swift` `SwiftUI` `SwiftData` `Supabase` `Anthropic API` `Cloudflare`
